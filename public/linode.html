Guide: Hosting a Website on a Dedicated Virtual Machine on Linode

Prerequisites

  1. An active Linode account.
  2. Basic knowledge of the tech stack you choose (Node.js, Python, Golang, or Rust).
  3. A code editor for development (e.g., Visual Studio Code).

Step 1: Create a Virtual Machine on Linode

  1. Sign in to Linode: https://www.linode.com/.
  2. Click on "Create" > "Linode" > "Add a Linode."
  3. Choose the appropriate options:
    • Linode plan: Select a plan that suits your website's needs.
    • Data center: Choose the data center location closest to your target audience.
    • Linux distribution: Select a distribution based on your preferred tech stack (e.g., Ubuntu for Node.js and Python, etc.).
    • Root password: Set a secure password for your VM.
  4. Click "Create" to deploy the VM.

Step 2: Access your Virtual Machine

  1. In the Linode Dashboard, go to "Linodes" and select your VM from the list.
  2. Click on "Remote Access" > "Launch Lish Console" to access the Linode Shell console.

Step 3: Install Tech Stack Dependencies

For Node.js:


# Update package lists
sudo apt update

# Install Node.js and npm
sudo apt install -y nodejs npm

For Python:


# Update package lists
sudo apt update

# Install Python and pip
sudo apt install -y python3 python3-pip

For Golang:


# Download and install Go
wget https://golang.org/dl/go1.XX.X.linux-amd64.tar.gz
sudo tar -C /usr/local -xzf go1.XX.X.linux-amd64.tar.gz
rm go1.XX.X.linux-amd64.tar.gz

# Add Go binaries to PATH (add this to your ~/.profile or ~/.bashrc)
export PATH=$PATH:/usr/local/go/bin

For Rust:


# Install Rust
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

# Add Rust binaries to PATH (this might be added automatically by the installation)
source $HOME/.cargo/env

Step 4: Deploy your Website

  1. Upload your website files to the Virtual Machine. You can use SCP or any other method you prefer.
  2. Place your website files in the appropriate directory based on the tech stack:
    • Node.js: Use a folder like /var/www/nodejs.
    • Python: Use a folder like /var/www/python.
    • Golang: Use a folder like /var/www/golang.
    • Rust: Use a folder like /var/www/rust.

Step 5: Run your Website

  1. Navigate to your website directory.
  2. Run the web server using the respective commands for each tech stack:

For Node.js:


# Install dependencies
npm install

# Start the server
node app.js

For Python:


# Install dependencies (if you have a requirements.txt)
pip install -r requirements.txt

# Start the server
python app.py

For Golang:


# Build and run the Go application
go run main.go

For Rust:


# Build and run the Rust application
cargo run

Step 6: Access Your Website

Your website should now be running on the dedicated virtual machine on Linode. Access it using the VM's public IP address.

For example:

  • http://YOUR_VM_PUBLIC_IP/

Remember to set up proper DNS records and domain configurations for a custom domain.

Congratulations! You have successfully hosted a website using Node.js, Python, Golang, or Rust on a dedicated virtual machine on Linode. Feel free to explore more advanced configurations and optimizations based on your website's needs. Happy coding!
